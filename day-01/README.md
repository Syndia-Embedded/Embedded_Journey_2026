# Day 01 — C Basics: Data Types & Sizes

## 📌 What I Practiced Today
- Understanding basic data types in C
- Using the `sizeof()` operator
- Printing memory sizes of variables
- Compiling and running a simple C program using GCC

## 📁 Files Included
- `code/sizes.c` — main C program for Day‑01
- `output/sizes-output.txt` — captured output from running the program
- `notes/day01-notes.md` — personal notes, learnings, and debugging steps

## 🧠 Key Learnings
- `sizeof()` returns the size of a variable or data type in bytes
- Data type sizes depend on system architecture (32‑bit vs 64‑bit)
- Importance of clean compilation using `gcc` and checking warnings
- How to redirect program output into a file
